## 🎯 Sobre o Projeto

Landing page completa para um Pet Shop, desenvolvida em HTML, CSS e JavaScript puro, com persistência de dados usando localStorage.

### ✅ Requisitos Atendidos

| Requisito | Status |
|-----------|--------|
| Seção de Destaque com position | ✅ |
| Seção Sobre com Flexbox/Grid | ✅ |
| Seção de Serviços com cards | ✅ |
| Formulário de contato com validação | ✅ |
| Responsividade | ✅ |
| Sistema de Agendamentos (Kanban) | ✅ |
| Persistência com localStorage | ✅ |

## 🚀 Como Executar

1. Baixe todos os arquivos
2. Mantenha na mesma pasta: `index.html`, `style.css`, `script.js`
3. Abra o arquivo `index.html` no navegador

## 📁 Arquivos

- `index.html` - Estrutura da página
- `style.css` - Estilos e responsividade
- `script.js` - Interatividade e lógica
- `README.md` - Documentação

## 🎨 Funcionalidades

- Agendamento de serviços
- Kanban com 3 colunas (Pendente/Em Andamento/Concluído)
- Mudança de status dos agendamentos
- Cancelamento de agendamentos
- Formulário de contato com validação
- Menu responsivo para mobile
- Animações e hover effects
